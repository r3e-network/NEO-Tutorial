## 概述

Neo 虚拟机 (NeoVM) 是一个轻量级的通用虚拟机，用于执行 Neo 智能合约。在 Neo N3 中，NeoVM 经过了显著优化，提供更好的性能、安全性和开发体验。

## Neo N3 主要特性

- **基于栈的架构** - 高效的执行模型
- **确定性执行** - 相同输入始终产生相同输出
- **沙箱环境** - 与主机系统隔离
- **丰富的指令集** - 超过 200 个操作码
- **互操作服务** - 访问区块链数据和原生合约
