# Gas 消耗

## 概述

NeoVM 中的每个操作都消耗 GAS，防止无限循环并确保公平的资源分配。

## 基础成本

| 类别 | 成本 (GAS) |
|------|------------|
| 简单操作 | 1/64000 |
| 算术运算 | 1/64000 |
| 栈操作 | 1/64000 |

## 存储成本

| 操作 | 成本 |
|------|------|
| Storage.Get | 0.01 GAS |
| Storage.Put (新建) | 0.1 GAS/字节 |
| Storage.Put (更新) | 0.01 GAS/字节 |
| Storage.Delete | 0.01 GAS |

## 优化建议

1. 减少存储写入
2. 批量操作
3. 使用合适的数据类型
4. 避免不必要的计算
