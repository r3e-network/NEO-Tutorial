# NeoVM 指令集

## 概述

NeoVM 使用超过 200 个操作码的丰富指令集。

## 指令分类

1. 常量操作
2. 流程控制
3. 栈操作
4. 算术操作
5. 逻辑操作

## 常量操作

| 操作码 | 名称 | 描述 |
|--------|------|------|
| 0x00 | PUSHINT8 | 压入 1 字节整数 |
| 0x01 | PUSHINT16 | 压入 2 字节整数 |
| 0x08 | PUSHT | 压入 true |
| 0x09 | PUSHF | 压入 false |

## 算术操作

| 操作码 | 名称 | 描述 |
|--------|------|------|
| 0x90 | ADD | 加法 |
| 0x91 | SUB | 减法 |
| 0x92 | MUL | 乘法 |
| 0x93 | DIV | 除法 |
| 0x94 | MOD | 取模 |
