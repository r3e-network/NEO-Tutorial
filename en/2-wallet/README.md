## Additional Resources

### Creating a wallet
There are several options available to you when creating a new wallet for Neo N3.

For full blockchain synchronization, consider:
* Neo-CLI -> https://docs.neo.org/docs/n3/node/cli/setup.html
* Neo-GUI -> https://docs.neo.org/docs/n3/node/gui/install.html

For light clients which do not require synchronization, consider:
* Neon Wallet -> https://neonwallet.com/
* NeoLine -> https://neoline.io/

### Neo N3 SDKs
* **C#**: [neo-devpack-dotnet](https://github.com/neo-project/neo-devpack-dotnet)
* **JavaScript/TypeScript**: [neon-js](https://github.com/CityOfZion/neon-js)
* **Python**: [neo-mamba](https://github.com/CityOfZion/neo-mamba)
* **Java**: [neow3j](https://github.com/neow3j/neow3j)
* **Go**: [neo-go](https://github.com/nspcc-dev/neo-go)

You can find more detailed usage guides at the relevant wallet links.
